# 노브랜드 버거 앱

### 주요 기능

> 앱에서 제공하는 기능 아래와 같습니다.
>> 1. 앱을 통해서 주문 기능
>> 2. 이벤트 및 공사사항 안내
>> 3. 회원 가입시 쿠폰 지급
>> 4. 회원일시 MY메뉴를 지정하여 편리하고 쉽게 주문
>> 5. 회원 스탬프를 통해서 10개 적립시 다양한 혜택
>> 6. 앱 환경설정
>> 7. 카트
